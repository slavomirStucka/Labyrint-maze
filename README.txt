# Labyrinth Game

## Overview
The **Labyrinth Game** is a browser-based game where players navigate through a maze. It uses a database to manage login credentials and track scores, with an interactive UI built using HTML and CSS.

## Features
- Player login system (with database integration)
- Score tracking and leaderboard
- Intuitive user interface using HTML, CSS, and JavaScript

## Usage
1. Sign up or log in to start playing.
2. Navigate the maze and try to reach the end in the shortest time!
